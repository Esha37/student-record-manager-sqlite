# Student Record Manager (SQLite)

## Overview

A console-based Student Record Manager developed using Python and SQLite. This application allows users to manage student records using a relational database.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Automatic Grade Calculation

## Technologies Used

- Python
- SQLite

## Skills Demonstrated

- SQLite Database
- SQL Queries
- CRUD Operations
- Functions
- Menu-Driven Programming
- Parameterized Queries

## How to Run

```bash
python student_record_manager.py
```
